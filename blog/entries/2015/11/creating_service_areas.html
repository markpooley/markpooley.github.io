Creating Service Areas from Point Data

November 18, 2015

Aggregating by Zip Code

Even if data aren't geocoded accurately, they can ge aggregated by zipcode or ZCTA accurately. First it's good to iterate through the data and look for zip codes that don't have a spatial counterpart (in the spatial data). A zip code to ZCTA crosswalk can be used for this. The goal is to take individual/point data and aggregate the number of visits origininating in each zip code or ZCTA and what zipcode the care was received in. The result is a table that looks like the following:

Recipient ZCTA Provider ZCTA Visits Total Visits Dyad Max
50001 50125 125 201 1
50001 50266 5 201 0

The "Dyad Max" variable serves as an indicator of which dyad pair the maxiumum number of visits occcured for all people originating in the "Recipient ZCTA".

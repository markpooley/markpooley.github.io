Working with loops

Working with loops and understanding how to structure them is how you leverage programming skills into working for you - reducing monotous work. You can loop through thousands of lines of data looking for specific occurences or data types in seconds. You can add conditional statements to loops, do loops within loops. We'll work with small lists to get the hang of working with loops. The structure is the same no matter if you have a list/array of 5 data points or 5 million.

import random
#empty list
ls = []

#create a boolean variable
addNumbers = True

#create your list of random numbers
while addNumbers:
  number = random.randint(0,20)
  ls.append(number)

  if len(ls) >= 20:
    addNumbers = False

def targetCounter(ls,target):
  counter = 0 #target counter/tracker variable
  #loop through list
  for i in ls:
    if i == target: #check if i is the target
      counter += 1 #increment counter

  #lets use a return statemetn here
  return counter

#call the function on your list of random numbers and assign it to a variable
#without a return statement in the targetCounter function this varible wouldn't
#actually have anything assigned to it.
